# Vernacular FD Advisor 💰

An AI-powered multilingual financial assistant that simplifies Fixed Deposits (FD) for beginners.

## 🚀 Live Demo
https://vernacular-fd-advisor-two.vercel.app/

## 🧠 Features
- AI chatbot (multi language support)
- FD calculator
- Personalized recommendations
- Simple booking simulation

## 🛠️ Tech Stack
- Frontend: React / Next.js
- Backend: Node.js / API routes
- AI: LLM integration
- ML: Rule-based + Decision Tree

## 📦 Installation

```bash
git clone https://github.com/birendra79/Vernacular_FD_Advisor.git
cd Vernacular_FD_Advisor
npm install
npm run dev
